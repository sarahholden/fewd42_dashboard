    Marty McFly Practice Exercise

    Complete before Tuesday's class

    Get some practice adding markup (HTML tags) to the Marty McFly About and Resume pages. You can find the copy and files here. In this download, an image has been provided for each page as a guide, along with the copy (text) for each page.

    Technical Requirements:

    • Add HTML tags to index.html
    • Add HTML tags to resume.html
    • Look up and implement the horizontal rule tag.
    • Look up and implement the nav tag.
    • Look up and implement the img tag.

    Bonus tasks:

    • Change the color of the anchors on hover (you'll have to look this one up)

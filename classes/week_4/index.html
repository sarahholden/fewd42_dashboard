    Dream Team

    Due June 19th at 11:59pm

    Your third assignment will be to finish the Dream Team we started in class (the version without Bootstrap).

    Technical Requirements:

    • Use floats in your CSS to achieve a two-column layout.
    • Use the correct Google Font typefaces to style the text denoted in the design file and according to the JPEG provided.
    • Use a background image for the header section of the page.
    • Add a hover effect to all the links and buttons using a pseudo-class.
    • Make use of clear:both and the clearfix method to make sure the footer stays at the bottom of the page.

    Bonus:

    Complete the advanced version of the site.

    Required Readings/Videos

    • Everyone: Read Smashing Magazine's article on Guidelines for Responsive Web Design.
    • If you purchased the textbook — HTML & CSS by Jon Duckett: Read pages 453 - 474 (Process)

    Be sure to browse through the GA project gallery if you haven't already and jot down a few ideas for your final project.

    Assignment 4

    Due June 19th at 11:59pm

    Your fourth assignment will be to finish the responsive version of the Dream Team lab we started in class.

    Technical Requirements:

    • Use media queries to achieve a responsive layout.
    • As the width of the browser window changes, layout should not look cramped, but columns should stack for smaller devices as needed.
    • Use rems/ems for all font sizes, as well as padding and margins.
    • Use percentages for all widths.

    Bonus:

    Customize the site to personalize the design.

    Add styles for the mobile nav when it is showing.

    Required Readings/Videos

    • If you purchased the textbook — HTML & CSS by Jon Duckett: Read pages 359 - 369 — Layout and positioning
    • If you purchased the textbook — HTML & CSS by Jon Duckett: Read pages 145 - 176 — Forms
    • Everyone: Read this article about CSS transitions: https://css-tricks.com/almanac/properties/t/transition/
